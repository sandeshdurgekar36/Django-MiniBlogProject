About This Project

Hello Viewers,

This Project is made Using Popular Web Framework(Django),

Project-Name:-

Mini-Blog

Basically I created This project to Add Post, Edit Post, and Delete Post. but the all the Permission such as add,delete,view,change are given only to the Admin Who can handle Everything.

AnyBody Who Wants To login (imported from AuthenticationForm provided by Django) only After filling the Form of Signup (imported from usercreationform provided by Django), they will get the permission of only Add and Edit the Post(Except Delete).

Usage of Authentication System :-

Another main concept is only Authenticated User can Access all the Authorizations Which is present inside the application(add,delete,view,change)

If anyone trying to access all the features before signup and login he/she directly redirected to login page
